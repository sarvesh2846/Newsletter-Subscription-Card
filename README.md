# 📧 Newsletter Subscription Card

A simple **newsletter subscription UI 📩** built with **Tailwind CSS 🎨** as a mini project to practice Tailwind concepts ⚡, utility classes, and modern UI styling ✨.

## 🚀 Features

- 📧 Newsletter subscription form
- 🎨 Gradient header design
- ✨ Clean and modern interface
- ⚡ Built entirely with Tailwind CSS
- 📚 Tailwind CSS learning project

## 🛠️ Tech Stack

- 🌐 HTML5
- 🎨 Tailwind CSS

## 📂 Project Structure

```bash
Newsletter-Subscription-Card/
│
├── index.html
├── style.css
├── package.json
├── package-lock.json
├── tailwind.config.js
├── postcss.config.js
└── README.md
```

## 🚀 Getting Started

```bash
git clone https://github.com/sarvesh2846/Newsletter-Subscription-Card.git

cd Newsletter-Subscription-Card

npm install

npm run dev
```

## 📚 Learning Outcome

This project was created while learning **Tailwind CSS** to practice utility classes, gradients, spacing, alignment, and component styling.
